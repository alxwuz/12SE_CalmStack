# CalmStack
A block game similar to Tetris with Python and customtkinter, only without time constraints and levels.

## Features
- Click to place mechanic on an 8x8 board
- User accounts with password hashing
- Saved high score
- Game over detection

## Setup + installation
```python
pip install -r requirements.txt
python main.py
# or, run it from any code editor.
```

## How to play
1. Log in or register an account.
2. Click a piece, then click the board to place it.
3. Filling a full row or column clears it for bonus points.
4. Game ends when no piece can be placed. Your high score saves automatically.
